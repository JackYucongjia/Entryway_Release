# Entryway Release Notes

## v0.8.1 - 2026-04-23

### 首页与服务器详情页加载优化

- **媒体库两段式加载**：首页和服务器详情页先快速展示前 6 个媒体库，随后后台异步补齐完整媒体库列表，避免首屏等待过久或后续媒体库完全不显示
- **精选轮播提速**：精选轮播不再强制等待 Logo 与背景图全部解析完成；背景图可用后即展示内容，Logo 后续加载补齐
- **轮播兜底展示优化**：当条目没有 Logo 或 Logo 加载较慢时，使用片名文字兜底，减少空白骨架停留时间
- **图片预取降噪**：收紧首页图片预取并发和首批预取数量，降低进入首页时的网络与解码竞争，让关键内容优先加载
- **精选数据轻量化**：精选轮播请求数量从 10 条缩减为 6 条，减少随机推荐和首屏图片解析成本

### 播放与调试链路收敛

- **起播诊断写盘关闭**：默认关闭 debug 包中的播放诊断文件写入，避免起播阶段产生额外 I/O
- **既有播放修复保留**：延续 v0.8.0 的 NextEmby 稳定流入口、认证参数补齐、Exo 外挂字幕修复和字幕浮窗文案修正
